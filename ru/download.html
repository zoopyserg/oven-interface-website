Подключение занимает 5 минут:

  • Arduino включить в USB
  • Модуль реле подключить в USB (вытащив из него джампер)
  • Подать 5V с Ардуино на плюсовую шину
  • Подать GND с Ардуино на минусовую шину
  • Присоединить пены Ардуино (с 3 по 10) на модуле Input в реле (порядок важен, настройки которое реле за что отвечает будут в интерфейсе).
  • Присоединить 5V с шины в VCC на реле модули.
  • Присоединить GND с шины в GND на реле модули.

Подключение термодатчика (на примере 3-проводной PT100, который уже есть в большинстве печей):

  • GND термодатчика в GND шину
  • Первый сигнальный провод (любой из двух которые остались) в аналоговый порт Ардуино (любой: А0-А5 - канал будет выбираться в Интерфейсные)
  • Второй сигнальный провод через резистор подключить к 5V шины.

После этого остается воткнуть через выбрано в Интерфейсные реле нагреватель, и все.

Термодатчик подавать температуру в интерфейс, а интерфейс будет поддерживать этим нагревателем нужную температуру.
